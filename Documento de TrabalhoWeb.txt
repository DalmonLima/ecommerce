Requisitos e observações - projeto de WEB 1Sistema de comércio eletrônico•Trata-se de um sistema para permitir comercialização de produtos por meio da web• Os produtos em estoque podem ser visualizados por todos os usuários• Deve haver dois perfis de usuário: administrador e comum.• Qualquer usuário (que tenha efetuado o login ou não) pode navegar pelo site e colocar, em seu carrinho de compra, os produtos que ele deseja comprar	– O carrinho de compra pode ser implementado com cookiesDeve ser informada a quantidade para cada produto adicionado no carrinho de compras• O usuário pode ter a opção de remover itens do carrinho de compra• O usuário só poderá efetuar a compra se estiver cadastrado e com uma sessão válida no sistema• O sistema deve ter um área para cadastro de novos usuários (área pública) e também a possibilidade do usuário alterar e excluir seus dados pessoais (área privada)• Somente o usuário administrador poderá modificar as quantidades dos produtos em estoque e visualizar todas as vendas realizadas no sitePrincipais recursos:• Cadastro de usuários, alteração dos dados de usuário e exclusão do usuário• Login e Logout da sessão de usuário• Carrinho de compras (pode ser implementado com cookies)• Realizar uma compra, caso existam produtos e quantidades disponíveis em estoque• Usuário com login válido poderá visualizar todas as suas compras e/ou realizar novas compras• Somente o usuário administrador possui uma interface para alterar as quantidades dos produtos em estoque e uma área de visualização de todas as vendas efetuadas no siteObservações:• Pesquise sites de compra na web para criar a interface do seu sistema• Utilize HTML, CSS e JavaScript• Atenção para as boas prática de uso• Deve ter sessão de usuário e cookiesEntrega• Aplicação web na versão final• Código-fonte e implantável• Documentação de instalação e de uso da aplicação que foi desenvolvida• O trabalho pode ser feito em equipes de no máximo três discentes• A avaliação será feita individualmenteTelas a serem implementadasE seus detalhesHome/Tela de navegaçãoPesquisarTela do produtoAdicionar ao
